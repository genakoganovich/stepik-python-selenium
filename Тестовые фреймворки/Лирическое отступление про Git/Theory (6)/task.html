Добавьте в репозиторий задачи из предыдущего модуля

Для закрепления материала предлагаем вам самостоятельно добавить файлы с решениями задач из предыдущего модуля в свой репозиторий на гитхабе.

Для этого вам нужно проделать следующее:

  1. Выберите один или несколько файлов с решениями задач из предыдущего модуля.
  2. Скопируйте их в папку с вашим локальным репозиторием.
  3. Добавьте файлы для отслеживания гитом.
  4. Выполните коммит (не забудьте добавить осмысленное сообщение!).
  5. Отправьте изменения на удаленный репозиторий.
  6. В браузере убедитесь, что файлы добавлены в репозиторий.
